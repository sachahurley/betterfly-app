Betterfly Onboarding - Data Export

View and export user responses from the onboarding flow

Setup Instructions:

  1. Create a new Google Sheet
  2. Copy the Google Apps Script code from google-apps-script.js
  3. In your Google Sheet, go to Extensions → Apps Script
  4. Replace the default code with the copied script
  5. Click Deploy → New Deployment
  6. Choose "Web app" as the type
  7. Set "Execute as" to "Me" and "Who has access" to "Anyone"
  8. Copy the Web App URL and paste it in data-collector.js where it says YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE
  9. To view data here, make your Google Sheet publicly viewable:
    • Click Share → Change to anyone with the link
    • Set permission to "Viewer"
    • Copy the sheet ID from the URL (between /d/ and /edit)
    • Paste it below to embed the sheet
